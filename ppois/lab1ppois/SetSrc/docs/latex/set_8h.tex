\doxysubsection{Detailed Description}
Множество, поддерживающее вложенность 

\begin{DoxyAuthor}{Author}
qudef 
\end{DoxyAuthor}
Класс {\ttfamily set} представляет из себя множество, способное содержать вложенные множества и целые числа вложенность реализуется с помощью библиотеки {\ttfamily std\+::variant}. Вложенные множества хрянятся в качестве {\ttfamily std\+::shared\+\_\+ptr\texorpdfstring{$<$}{<}set\texorpdfstring{$>$}{>}}. Поддерживает операции пересечения, объединения, разности и булеана. Множество может быть задано задано через строку, массив типа char или через std\+::initializer\+\_\+list.
